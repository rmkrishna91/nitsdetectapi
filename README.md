# NITS Detect API

An AI text detection API built with FastAPI, PyTorch, and Sentence
Transformers. The API analyzes a given text and predicts whether it is
likely generated by AI or written by a human.

## Features

-   AI vs Human text detection
-   REST API built with FastAPI
-   Sentence embeddings using all-mpnet-base-v2
-   Linguistic feature extraction (POS tags, stopwords, etc.)
-   Docker support for easy deployment
-   Swagger UI for API testing

## Installation

### 1. Clone the repository

``` bash
git clone https://github.com/rmkrishna91/nitsdetectapi.git
cd nitsdetectapi
```

### 2. Create virtual environment

``` bash
python3 -m venv venv
source venv/bin/activate
```

### 3. Install dependencies

``` bash
pip install -r requirements.txt
```

### 4. Install spaCy model

``` bash
python -m spacy download en_core_web_sm
```

## Running the API

Start the FastAPI server:

``` bash
uvicorn app.main:app --host 0.0.0.0 --port 8000
```

Open in browser:

    http://localhost:8000/docs

This launches Swagger UI where you can test the API.

------------------------------------------------------------------------

## API Endpoint

### POST /predict

Request body:

``` json
{
  "text": "Artificial intelligence is transforming industries."
}
```

Response:

``` json
{
  "prediction": 1,
  "probability": 0.87
}
```

Prediction values:

    0 → Human written
    1 → AI generated

## Streamlit UI

Create `streamlit_app.py`:

``` python
import streamlit as st
import requests

API_URL = "http://localhost:8000/predict"

st.title("AI Text Detection")

st.write("Enter text to check if it is AI-generated or human-written.")

text_input = st.text_area("Enter your text here")

if st.button("Analyze"):

    if text_input.strip() == "":
        st.warning("Please enter some text.")
    else:

        response = requests.post(
            API_URL,
            json={"text": text_input}
        )

        if response.status_code == 200:

            result = response.json()

            prediction = result["prediction"]
            probability = result["probability"]

            if prediction == 1:
                st.error(f"AI Generated (Confidence: {probability:.2f})")
            else:
                st.success(f"Human Written (Confidence: {probability:.2f})")

        else:
            st.error("API request failed.")
```

Run Streamlit:

``` bash
pip install streamlit
streamlit run streamlit_app.py
```

Open:

    http://localhost:8501
